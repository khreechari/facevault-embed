# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

In short: be respectful. Disagree without being disagreeable. We're all here
to ship reliable identity verification, not to win arguments.

## Reporting

If someone in this community is acting in a way that violates the Contributor
Covenant, email **conduct@facevault.id**. Reports are handled confidentially.
We commit to a response within 5 business days.

The maintainers reserve the right to remove, edit, or reject comments,
commits, code, issues, and other contributions that violate this code of
conduct.
